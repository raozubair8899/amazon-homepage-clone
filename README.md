# Amazon-Homepage-Clone-HTML5-CSS3

A responsive clone of the **Amazon** homepage built with **HTML5** and **CSS3** (vanilla CSS, no frameworks).  

This project recreates the classic Amazon layout including the navigation bar, search functionality, hero banner, product category grid, and footer â€” perfect for practicing real-world UI replication, flexbox, responsive design, and hover effects.

## Features

- Sticky dark navigation bar with logo, delivery address, search bar, account links, returns, and cart
- Category panel ("All", Today's Deals, etc.)
- Full-width hero banner
- Responsive grid of product category cards (Home, Health, Furniture, Electronics, Beauty, etc.)
- Multi-section footer with links, logo, and copyright
- Hover border effects on interactive elements
- Mobile-friendly layout (flex-wrap + percentage widths)
